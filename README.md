# Machine Learning Projects

A collection of machine learning notebooks and datasets covering fundamental algorithms and classification techniques.

## Notebooks

- **GDA.ipynb** - Gaussian Discriminant Analysis implementation and analysis
- **linear_regression.ipynb** - Linear regression models and techniques
- **logistic_regression.ipynb** - Logistic regression for binary classification
- **naive_bayes.ipynb** - Naive Bayes classifier implementation
- **perceptron.ipynb** - Perceptron learning algorithm
- **softmax_regression.ipynb** - Softmax regression for multi-class classification

## Datasets

- **placements.csv** - Placement prediction dataset
- **Salary_dataset.csv** - Salary prediction dataset
- **shapes.csv** - Shape classification dataset
- **spam.csv** - Spam email classification dataset
- **student_pass.csv** - Student pass/fail prediction dataset
- **tumor.csv** - Tumor classification dataset

## Getting Started

1. Install dependencies:
   ```bash
   pip install jupyter pandas numpy scikit-learn matplotlib
   ```

2. Activate the virtual environment:
   ```bash
   .\venv\Scripts\Activate.ps1
   ```

3. Open a notebook:
   ```bash
   jupyter notebook
   ```

## Requirements

- Python 3.x
- Jupyter Notebook
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
